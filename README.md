基于原生前端重构的氛围感心情日记，LocalStorage 本地离线存储，无后端零依赖，全新自定义视觉交互改版，轻量化情绪记录小工具

## 项目概述
本项目是治愈系本地心情记录单页应用，有纯净Vibe coding原版与修订版。
修订版基于原版基础做了全套个人风格视觉、交互逻辑修订优化。
全程仅单 HTML 文件，无任何框架、打包工具、后端服务，打开即用；所有日记数据仅存储在浏览器本地 LocalStorage，隐私数据不会上传网络。
仓库内置两套完整界面实现：
1. diary_org原版：低饱和马卡龙磨砂玻璃治愈UI
2. moodiary修订版：深色终端赛博复古风格（截图展示版本）

## 原版核心功能
- 五档心情快速选择：低落 / 平淡 / 普通 / 开心 / 狂喜
- 自由文本日记撰写，记录日常碎碎念
- 本地持久化存储，刷新页面数据不丢失
- 时间轴倒序展示全部历史日记
- 单条记录一键删除
- 对应心情自动匹配专属柔和背景配色
- 全端响应式，手机、电脑均可流畅使用

## 🎮 修订版（开头大图）新增/改动特性

### 本次改版修订亮点
- 深色暗紫终端底色，荧光绿色终端风格文字，复古控制台氛围
- 每种心情对应专属暗调色块，替代原版浅马卡龙底色
- 去掉毛玻璃柔光，改用硬朗复古控制台矩形区块、立体按钮边框
- 整体低亮度暗色主题，护眼夜间使用

### 交互优化调整
- 心情选项扩充为6档：低落、平淡、普通、不选、快乐、狂喜，搭配复古表情符号
**注意:可能与原版不兼容**
- 历史记录条目深色分区，删除按钮高对比红色标识
- 布局结构重新分层：输入区→操作提示栏→历史记录分区，控制台层级逻辑
- 适配暗色屏幕观感，文字对比度重新调校

### 新增核心备份功能（原版无）
1. **导出Base64日记**：全部记录编码为Base64字符串，一键复制到剪贴板跨设备迁移
2. **复制明文到剪贴板**：一键导出所有日记纯文本，快速备份
3. 全局消息提示栏：操作反馈统一控制台式提示（导出成功/保存完成等）
